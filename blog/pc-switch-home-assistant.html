Building a remote PC Switch compatible with Home Assistant

Github Repository

Introduction

The goal of this project is to have a remote switch for my personal workstation, which can be triggered from Home Assistant.

Components

  • Raspberry Pi 3B+
  • Docker Host (Thinkcentre M-Series)
  • Relay Module | SRD-05vdc-sl-c (Currently operated under voltage [need to buy another one])
  • A few jumper cable

Software

The software running on the Raspberry Pi is written in Rust with a custom implementation for the GPIO pins. The Docker host has a very basic Python script to send packets to the TCP socket. Because there is ony one relay which is controlled the server only accepts a single connection at a time.

The code for the Rust TCP Server can be found on my Github (pc-switch-rs) and the Python script, which is executed by Home Assistant, is down below.


import socket

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect(("raspberrypi", 8000))
s.sendall(b'500\n')
s.close()
